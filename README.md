##### #自定义js函数，包括手写函数以及常用工具库

### 安装 

yarn add js-tools-function

### 或者

npm install js-tools-function



### 使用

需要开启服务才可以使用

```javascript
import fun from 'js-tools-function'
```



### 目前包含的函数有

##### 		改变this指向

2. apply(fn , obj , args)
